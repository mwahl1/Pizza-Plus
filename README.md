# Pizza-Plus

Project: SWDV 116 Project #2 - Styling Your Business Site
This project is a simple multi-page website for a fictional pizza restaurant.

Pages

1. index.html – Home page with an introduction and image.
2. menu.html – Menu items and prices.
3. specials.html – Weekly specials and deals.
4. contact.html – Contact information and a fully styled contact form.
5. about.html – Background information about Pizza Plus.

All pages share a common navigation bar and use the same style.css for consistent branding.

Styles

- Main stylesheet: style.css
- No inline CSS is used. All styling is handled through the external CSS file.
